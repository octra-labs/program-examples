# program examples

simple AML program examples for Octra.

these programs are written in AppliedML (`.aml`), Octra's high-level program language.
AML programs define state, entrypoints, checks, events, and constructors, then compile to OCTB bytecode for the Octra VM.

docs: [introduction to applied](https://docs.octra.org/developer-docs/introduction-to-applied)

- amm
- empty
- escrow
- multisig
- private_ml
- token
- vault
